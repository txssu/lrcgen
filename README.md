# lrcgen

> 100% of the code in this repo was generated by AI. If that bothers you, save yourself the time.

TUI for making LRC files. Play a song, tap spacebar on each line, get timestamps. Fix what's off, save.

## Install

### Binary (no dependencies)

Grab the binary for your OS from [Releases](https://github.com/txssu/lrcgen/releases). You still need `ffplay` (part of [ffmpeg](https://ffmpeg.org/download.html)) on your system for audio playback.

### From source

Needs [Bun](https://bun.sh) and [ffmpeg](https://ffmpeg.org/download.html).

```bash
git clone https://github.com/txssu/lrcgen.git
cd lrcgen
bun install
bun run start
```

## Architecture

Port/adapter pattern. Audio sources, lyrics providers, LRC parser, and lyrics publishers are swappable interfaces. Ships with ffplay for audio, clipboard/LRCLIB for lyrics, and LRCLIB for publishing.
